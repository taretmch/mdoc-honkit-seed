mdoc のサンプルコード

以下のようなコードを書くと、mdoc によってコンパイルされ、式の評価結果が表示されます。

before:

val x = 1
List(x, x)

after:

val x = 1
// x: Int = 1
List(x, x)
// res0: List[Int] = List(1, 1)

また、import も同一ファイル内で有効なので、

import scala.util.Try

このように import すれば、別のコードブロックにおいても使用することができます。

val y = Try(3)
// y: Try[Int] = Success(3)
val z = Try(1 / 0)
// z: Try[Int] = Failure(java.lang.ArithmeticException: / by zero)
